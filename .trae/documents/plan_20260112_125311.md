## 商家发货信息设置功能实现计划

### 1. 项目结构创建
- 在项目根目录创建 `proto` 目录，用于存放原型设计文件

### 2. 高保真原型设计
已完成 `proto/shipping-settings.html` 高保真原型，包含以下功能模块：

#### 2.1 基本发货信息
- 发货人姓名、联系电话、详细地址
- 省份、城市、区/县三级联动选择
- 邮政编码和发货仓库选择

#### 2.2 物流商设置
- 物流商列表管理（名称、编码、联系人、电话）
- 默认物流商设置
- 物流商添加、编辑、删除功能

#### 2.3 发货模板设置
- 模板名称和描述
- 默认物流商和重量设置
- 模板添加、编辑、删除功能

### 3. 技术实现特点
- 采用 Element Plus 组件库，保持与现有项目风格一致
- 响应式布局设计，适配不同屏幕尺寸
- 完整的表单验证和交互逻辑
- 模拟数据展示，便于直观了解功能

### 4. 后续开发步骤
- 根据原型图实现 Vue 3 组件
- 集成到商家端菜单路由
- 实现与后端 API 的数据交互
- 添加表单验证和错误处理
- 测试发货信息在订单发货流程中的应用
